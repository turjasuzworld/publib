This is a standard communication driver designed for AT command set communication with ESP01 type of WIFI modems. The various low level api that has been designed are there to establish provisioning with the wifi network, communicate with it for REST api and evaluate the RSSI. It will be elaborated as and when we develop and use other features.
If you want to contribute you are free to do so. Please inform me at turjasu@outlook.com before pushing your self tested code and it can break the functionality. Donot push codes with active / possible bugs in it!
